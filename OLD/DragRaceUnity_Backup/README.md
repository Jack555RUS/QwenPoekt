СЦЕНЫ:
================================================================================

1. Start.unity
   - Canvas с кнопкой "START" (200x60, центр)
   - StartSceneController → при клике загружает MainMenu
   
2. MainMenu.unity
   - Canvas с 3 кнопками (200x50, вертикально):
     * NEW GAME → загружает Race.unity
     * CONTINUE → (заглушка)
     * EXIT → закрывает игру
   - MainMenuController подключён к кнопкам
   
3. Race.unity
   - Canvas с 3 кнопками + текст счётчика:
     * Сменить цвет (180x45) → меняет цвет камеры
     * Накрутить счётчик (180x45) → +1 к счётчику
     * В меню (180x45) → возврат в MainMenu
   - CounterLabel: "Счётчик:" (слева)
   - CounterValue: "0" (справа, обновляется)
   - RaceManager подключён к кнопкам и тексту

ДЛЯ ЗАПУСКА В UNITY EDITOR:
================================================================================

1. Unity Hub → Add → Выбрать папку:
   D:\QwenPoekt\ProbMenu\DragRaceUnity

2. Открыть проект в Unity 6000.3.10f1

3. Открыть сцену: Assets/Scenes/Start.unity

4. Нажать Play (Ctrl+P)

5. Проверить работу:
   - Клик START → переход в MainMenu
   - Клик NEW GAME → переход в Race
   - Клик "Сменить цвет" → фон меняется
   - Клик "Накрутить счётчик" → число растёт
   - Клик "В меню" → возврат в MainMenu
   - Клик EXIT → закрытие

================================================================================
ДЛЯ СБОРКИ В EXE:
================================================================================

1. В Unity Editor: File → Build Settings

2. Добавить сцены (Add Open Scenes):
   - Assets/Scenes/Start.unity  (индекс 0)
   - Assets/Scenes/MainMenu.unity (индекс 1)
   - Assets/Scenes/Race.unity   (индекс 2)

3. Platform: Windows, Mac, Linux
   Target Platform: Windows
   Architecture: x86_64

4. Player Settings:
   - Company Name: ProbMenu Studio
   - Product Name: ProbMenu
   - Resolution: 640x480

5. Нажать "Build"

6. Выбрать папку: D:\QwenPoekt\ProbMenu\DragRaceUnity\Builds

7. Дождаться сборки (~2-5 минут)

8. Запустить: Builds/ProbMenu.exe
