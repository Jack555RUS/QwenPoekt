  - LeftShift / N - Нитро

ИНСТРУКЦИЯ ПО СБОРКЕ EXE:
================================================================================

1. Открыть проект в Unity:
   - Unity Hub → Add → D:\QwenPoekt\ProbMenu\DragRaceUnity
   - Открыть в Unity 6000.3.10f1

2. Проверить компиляцию:
   - Window → General → Console
   - Убедиться что ошибок нет

3. Настроить Build Settings:
   - File → Build Settings
   - Platform: Windows, x86_64
   - Resolution: 800x600
   - Fullscreen Mode: Windowed

4. Добавить сцены:
   - Assets/Scenes/MainMenu.unity (индекс 0)
   - Assets/Scenes/GameMenu.unity (индекс 1)
   - Assets/Scenes/Race.unity (индекс 2)

5. Сборка:
   - Нажать "Build"
   - Выбрать папку: D:\QwenPoekt\ProbMenu\Builds\
   - Имя файла: DragRace.exe

6. Тестирование:
   - Запустить DragRace.exe
   - Проверить все меню
   - Провести гонку
