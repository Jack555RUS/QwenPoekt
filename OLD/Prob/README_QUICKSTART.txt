================================================================================
                        RACING GAME - QUICK START
================================================================================

OPTION 1: Automatic Build (EASIEST!)
-------------------------------------

1. Double-click: BuildGame.bat

2. Wait for Unity to build the game (5-10 minutes)

3. Open "Build" folder

4. Run: RacingGame.exe

5. Play!


OPTION 2: Unity Editor
-----------------------

1. Open project in Unity (version 6000.3.10f1 or newer)

2. Menu: RacingGame → Quick Setup and Build

3. Wait for build to complete

4. Open "Build" folder

5. Run: RacingGame.exe


OPTION 3: Manual Build in Unity
--------------------------------

1. Open project in Unity

2. Wait for compilation to finish

3. File → Build Settings

4. Click "Build"

5. Choose folder: D:\QwenPoekt\Prob\Build

6. Run: RacingGame.exe


================================================================================
                        CONTROLS
================================================================================

MAIN MENU:
  - New Game     - Create new game
  - Continue     - Load saved game
  - Settings     - Game settings
  - Exit         - Quit game

KEYBOARD:
  W, A, S, D    - Move
  Enter         - Confirm
  Escape        - Back / Pause
  S             - Save
  L             - Load


================================================================================
                        SYSTEM REQUIREMENTS
================================================================================

MINIMUM:
  OS: Windows 10 64-bit
  CPU: Intel Core i3
  RAM: 4 GB
  GPU: DirectX 11 compatible
  Disk: 500 MB


================================================================================
                        TROUBLESHOOTING
================================================================================

ERROR: Unity not found
  → Install Unity 6000.3.10f1 or newer
  → https://unity.com/download

ERROR: Build failed
  → Open project in Unity manually
  → Check Console for errors
  → Fix errors and try again

ERROR: Game won't start
  → Install latest .NET Framework
  → Run as Administrator
  → Check antivirus
