=====================================================================
ИНСТРУКЦИЯ ПО НАСТРОЙКЕ СЦЕНЫ MAINMENU
=====================================================================

Эта инструкция поможет вам правильно настроить сцену главного меню
для проекта Racing Game.

=====================================================================
ШАГ 1: ОТКРЫТЬ СЦЕНУ
=====================================================================

1. В Unity Editor откройте файл:
   D:\QwenPoekt\Prob\Assets\Scenes\MainMenu.unity

2. Вы должны увидеть пустую сцену с камерой (Main Camera)

=====================================================================
ШАГ 2: СОЗДАТЬ ОБЪЕКТ INITIALIZER
=====================================================================

1. В окне Hierarchy нажмите правой кнопкой мыши
2. Выберите: Create Empty (Создать пустой объект)
3. Переименуйте его в: MenuInitializer

4. Выделите MenuInitializer и в Inspector нажмите:
   Add Component → MainMenuInitializer

5. Заполните поля компонента:
   
   [Main Menu Asset]
   → Перетащите файл: Assets/UI Toolkit/MainMenu.uxml
   
   [Settings Menu Asset]
   → Перетащите файл: Assets/UI Toolkit/SettingsMenu.uxml
   
   [Audio Mixer]
   → Перетащите файл: Assets/AudioMixer.mixer

=====================================================================
ШАГ 3: НАСТРОИТЬ КАМЕРУ
=====================================================================

1. Выделите объект: Main Camera

2. В компоненте Camera установите:
   
   Projection: Orthographic
   Clear Flags: Solid Color
   Background: R=0.1, G=0.1, B=0.1, A=1 (темно-серый)
   Culling Mask: Everything
   Field of view: 60

=====================================================================
ШАГ 4: ПРОВЕРИТЬ BUILD SETTINGS
=====================================================================

1. Нажмите: File → Build Settings (Ctrl+Shift+B)

2. В списке "Scenes In Build" должны быть:
   ☑ Assets/Scenes/MainMenu.unity (индекс 0)
   ☑ Assets/Scenes/Game.unity
   ☑ Assets/Scenes/Garage.unity
   ☑ Assets/Scenes/Tuning.unity
   ☑ Assets/Scenes/Shop.unity

3. Если сцен нет - перетащите их из окна Project

=====================================================================
ШАГ 5: НАСТРОЙКИ PLAYER
=====================================================================

1. В Build Settings нажмите "Player Settings"

2. В разделе "Resolution and Presentation":
   Default Screen Width: 800
   Default Screen Height: 600
   Fullscreen Mode: Windowed
   Run In Background: ✓

3. В разделе "Other Settings":
   Company Name: DefaultCompany
   Product Name: RacingGame
   Version: 1.0.0

=====================================================================
ШАГ 6: ПРОВЕРКА QUALITY SETTINGS
=====================================================================

1. Откройте: Edit → Project Settings → Quality

2. Должны быть уровни:
   Level 0: Low
   Level 1: Medium
   Level 2: High
   Level 3: Ultra

3. По умолчанию выбран: Medium (Level 1)

=====================================================================
ШАГ 7: ЗАПУСК И ПРОВЕРКА
=====================================================================

1. Нажмите кнопку Play (▶) в Unity Editor

2. Проверьте работу кнопок:
   
   ✓ Главное меню отображается
   ✓ Кнопка "НАСТРОЙКИ" открывает меню настроек
   ✓ Ползунки громкости работают
   ✓ Dropdown разрешения заполняется
   ✓ Toggle fullscreen переключается
   ✓ Кнопка "НАЗАД" возвращает в главное меню
   ✓ Кнопка "ВЫХОД" останавливает игру (в Editor)

3. Проверьте логи:
   Window → General → Console
   Или файлы в: D:\QwenPoekt\Prob\Logs\

=====================================================================
ДОПОЛНИТЕЛЬНАЯ НАСТРОЙКА
=====================================================================

Если кнопки не работают:

1. Проверьте, что имена кнопок в UXML совпадают:
   - NewGameButton
   - ContinueButton
   - SaveButton
   - LoadButton
   - SettingsButton
   - ExitButton
   - BackButton (в настройках)
   - MusicVolumeSlider
   - SFXVolumeSlider
   - FullscreenToggle
   - ResolutionDropdown
   - QualityDropdown

2. Проверьте логи на ошибки компиляции

3. Убедитесь, что все менеджеры созданы:
   - GameManager
   - MenuManager
   - AudioManager
   - InputManager

=====================================================================
СОЗДАНИЕ БИЛЬДА
=====================================================================

1. File → Build Settings

2. Platform: Windows, Mac, Linux Standalone
   Target Platform: Windows
   Architecture: x86_64

3. Нажмите "Build"

4. Выберите пустую папку для сборки

5. После сборки запустите .exe файл и проверьте работу

=====================================================================
КОНТРОЛЬНЫЙ СПИСОК
=====================================================================

Перед запуском проверьте:
☐ MainMenuInitializer создан и настроен
☐ Все ссылки на UXML файлы заполнены
☐ AudioMixer подключен
☐ Камера настроена (Orthographic, Solid Color)
☐ Сцены добавлены в Build Settings
☐ Проект компилируется без ошибок
☐ Логи не содержат ошибок
