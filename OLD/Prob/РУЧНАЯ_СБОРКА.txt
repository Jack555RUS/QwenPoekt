================================================================================
              RACING GAME - РУЧНАЯ СБОРКА (если автоматическая не работает)
================================================================================

ШАГ 1: Откройте проект в Unity
-------------------------------

1. Запустите Unity Hub

2. Нажмите "Add" (Добавить проект)

3. Выберите папку: D:\QwenPoekt\Prob

4. Дождитесь импорта проекта (может занять 5-10 минут)


ШАГ 2: Проверьте ошибки компиляции
-----------------------------------

1. Откройте Console: Window → General → Console

2. Если есть ошибки (красные) - исправьте их

3. Дождитесь пока исчезнет надпись "Compiling..." внизу


ШАГ 3: Настройте сцены
-----------------------

1. Откройте: File → Build Settings (Ctrl+Shift+B)

2. Добавьте сцены кнопкой "Add Open Scenes" или перетащите:
   ☑ Assets/Scenes/MainMenu.unity
   ☑ Assets/Scenes/Game.unity
   ☑ Assets/Scenes/Garage.unity
   ☑ Assets/Scenes/Tuning.unity
   ☑ Assets/Scenes/Shop.unity


ШАГ 4: Соберите проект
----------------------

1. В Build Settings нажмите "Build"

2. Создайте папку "Build" в корне проекта

3. Выберите файл: Build\RacingGame.exe

4. Нажмите "Save"

5. Дождитесь завершения сборки (5-15 минут)


ШАГ 5: Запустите игру
---------------------

1. Откройте папку: D:\QwenPoekt\Prob\Build

2. Запустите: RacingGame.exe

3. Играйте!


================================================================================
                      ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
================================================================================

ПРОБЛЕМА: Unity не открывает проект
РЕШЕНИЕ: Установите Unity 6000.3.10f1 или новее
         https://unity.com/download

ПРОБЛЕМА: Ошибки компиляции в Console
РЕШЕНИЕ: 
  1. Проверьте что все .asmdef файлы имеют правильные ссылки
  2. Удалите папки Library и Temp
  3. Перезапустите Unity

ПРОБЛЕМА: Build не создается
РЕШЕНИЕ:
  1. Проверьте что все сцены добавлены в Build Settings
  2. Проверьте что нет ошибок компиляции
  3. Попробуйте собрать пустую сцену сначала

ПРОБЛЕМА: RacingGame.exe не запускается
РЕШЕНИЕ:
  1. Установите .NET Framework последней версии
  2. Запустите от имени администратора
  3. Проверьте антивирус


================================================================================
                      БЫСТРАЯ ПРОВЕРКА ПРОЕКТА
================================================================================

Перед сборкой убедитесь:

☐ Проект открывается в Unity без ошибок
☐ В Console нет красных ошибок
☐ Все 5 сцен существуют в Assets/Scenes/
☐ Менеджеры работают (GameManager, MenuManager, etc.)
☐ Тесты проходят (Window → Test Runner → Run All)


================================================================================

Если ничего не помогает:

1. Откройте проект в Unity
2. Дождитесь окончания компиляции
3. Нажмите Play (▶) чтобы проверить работу в Editor
4. Если работает - делайте сборку через File → Build Settings
