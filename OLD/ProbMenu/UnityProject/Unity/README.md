# 2D Меню для Unity

## Структура проекта
```
Unity/
├── Assets/
│   ├── Scripts/
│   │   ├── MenuController.cs      # Логика кнопок
│   │   └── SetupUIScene.cs        # Автоматическая настройка UI
│   └── Scenes/
│       └── MainMenu.unity         # Основная сцена
```

## Инструкция по запуску

### Вариант 1: Автоматическая настройка
1. Откройте Unity Hub
2. Создайте новый 2D проект
3. Скопируйте папку `Assets` в проект
4. Откройте сцену `MainMenu.unity`
5. Нажмите Play

### Вариант 2: Ручная настройка
1. Создайте новый 2D проект в Unity
2. Создайте папку `Scripts` в `Assets`
3. Скопируйте скрипты из `Assets/Scripts/`
4. Создайте пустой GameObject на сцене
5. Добавьте компонент `SetupUIScene`
6. Нажмите Play (скрипт автоматически создаст UI)

## Элементы управления

| Кнопка | Действие |
|--------|----------|
| Сменить цвет | Меняет цвет фона камеры на случайный |
| Накрутить счётчик | Увеличивает счётчик на 1 |
| Выход | Закрывает приложение |

## Настройки сцены
- Разрешение: 640x480
- Камера: Orthographic
- UI: Canvas (Screen Space - Overlay)
