## 🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### ❌ Игра не запускается

**Решение:**
```
1. Проверьте что все файлы в папке
2. Запустите от имени администратора
3. Проверьте антивирус
4. Установите DirectX End-User Runtimes
```

### ❌ Чёрный экран

**Решение:**
```
1. Нажмите Alt+Enter (полноэкранный режим)
2. Проверьте совместимость видеокарты
3. Обновите драйверы
```

### ❌ Звук не работает

**Решение:**
```
1. Проверьте настройки Windows
2. Нажмите Esc → Settings → Audio
3. Увеличьте громкость
```
