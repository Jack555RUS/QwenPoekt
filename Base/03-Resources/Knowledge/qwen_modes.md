## 🎯 НАЗНАЧЕНИЕ

Описание режимов работы Qwen 3.5 Plus и их правильное использование.

---

## 📋 РЕЖИМЫ

### 1. Режим принятия правок (Tab)

**Назначение:** Предложения кода с подтверждением

**Когда использовать:**
- ✅ Повседневная разработка
- ✅ Рефакторинг
- ✅ Исправление багов

**Как работает:**
```
ИИ предлагает → Вы нажимаете Tab → Применяется
```

**Пример:**
```
Запрос: "Добавь проверку на null в метод GetPlayer"

ИИ предлагает:
```csharp
public Player GetPlayer()
{
    if (_player == null)
    {
        Debug.LogWarning("Player is null");
        return null;
    }
    return _player;
}
```

Вы нажимаете Tab → Код добавляется
```

---

### 2. Режим YOLO

**Назначение:** Автономное применение изменений

**Когда использовать:**
- ✅ Массовое форматирование
- ✅ Простые изменения
- ✅ Переименование

**Предупреждение:**
> ⚠️ Изменения применяются автоматически!

**Пример:**
```
Запрос: "Замени все var на явные типы в файле Program.cs"

ИИ выполняет:
- Находит все `var`
- Заменяет на явные типы
- Сохраняет файл без подтверждения
```

---

### 3. Режим планирования

**Назначение:** План действий перед выполнением

**Когда использовать:**
- ✅ Сложные многошаговые задачи
- ✅ Архитектурные изменения
- ✅ Новые функции

**Как работает:**
```
Запрос → План → Проверка → Выполнение
```

**Пример:**
```
Запрос: "Добавь систему сохранений в игру"

ИИ составляет план:
1. Создать класс SaveData
2. Добавить метод Save()
3. Добавить метод Load()
4. Интегрировать в GameManager

Вы проверяете план → ИИ выполняет по шагам
```

---

### 4. Обычный режим (фиолетовый >)

**Назначение:** Ввод команд и вопросов

**Когда использовать:**
- ✅ Вопросы
- ✅ Короткие команды
- ✅ Объяснения

**Пример:**
```
> объясни что такое Dependency Injection

ИИ отвечает:
Dependency Injection — это паттерн проектирования...
```

---

## 📊 СРАВНЕНИЕ

| Задача | Режим | Почему |
|--------|-------|--------|
| Написание кода | Принятия правок | Контроль изменений |
| Форматирование | YOLO | Экономия времени |
| Сложная задача | Планирования | Стратегия перед кодом |
| Вопрос | Обычный | Быстрый ответ |
