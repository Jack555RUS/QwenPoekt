## 📖 Извлечённая информация

**Основная идея:** Автоматический бэкап перед операциями

**Ключевые функции:**
- `Get-File-Count` — подсчёт файлов
- `Get-Folder-Size-MB` — вычисление размера
- `Test-Backup-Integrity` — проверка целостности

**Параметры:**
- `-OperationType` (обязательный) — тип операции
- `-BackupRoot` (необязательный) — путь к бэкапам
- `-WhatIf` (флаг) — режим проверки

**Пример использования:**
```powershell
.\scripts\pre-operation-backup.ps1 -OperationType "Delete_OldProject"
```

**Предупреждения:**
- Требуется место на диске (~50-200 MB)
- Время выполнения: 2-5 минут

## 📖 Извлечённая информация

**Симптомы:**
- Потеряно ~21 скрипт PowerShell
- Потеряно ~37 отчётов
- Причина: досрочное закрытие терминала

**Причина:**
- Работа без бэкапа
- Прямое удаление из _TEMP
- Нет журнала операций

**Решение:**
```powershell
# 1. Восстановление из Git
git restore Base/03-Resources/PowerShell/
git restore Base/reports/

# 2. Проверка целостности
.\scripts\check-duplicates.ps1

# 3. Создание бэкапа
.\scripts\pre-operation-backup.ps1 -OperationType "Restore"
```

**Проверка успеха:**
```powershell
# Проверить количество файлов
(Get-ChildItem Base/scripts -Recurse -File).Count
# Ожидается: 33+
```
