# Настройка Qwen 3.5 Pro в VS Code

**Версия:** 1.0  
**Дата:** 2026-03-02  
**Статус:** ✅ Готово

---

## 🎯 НАЗНАЧЕНИЕ

Настройка Qwen 3.5 Pro для работы в Visual Studio Code с максимальными возможностями.

---

## 📋 ТРЕБОВАНИЯ

- ✅ Visual Studio Code 2026
- ✅ Учётная запись GitHub
- ✅ Подписка на Qwen (или бесплатный тариф)
- ✅ Расширение Qwen Code для VS Code

---

## 🔧 УСТАНОВКА

### Шаг 1: Установка расширения

1. Откройте VS Code
2. Перейдите в **Extensions** (`Ctrl+Shift+X`)
3. Найдите **Qwen Code**
4. Нажмите **Install**

---

### Шаг 2: Авторизация

1. Нажмите `Ctrl+Shift+P` → введите **Qwen: Sign In**
2. Войдите через GitHub
3. Подтвердите доступ

---

### Шаг 3: Настройка режимов

**Режимы Qwen 3.5 Plus:**

| Режим | Клавиша | Назначение |
|-------|---------|------------|
| **Принятия правок** | `Tab` | Предложения с подтверждением |
| **YOLO** | `Ctrl+Y` | Автономное применение |
| **Планирования** | `Ctrl+P` | План перед выполнением |
| **Обычный** | `>` | Ввод команд |

---

## ⚙️ НАСТРОЙКИ

### Рекомендуемые параметры:

```json
{
  "qwen.model": "qwen-3.5-plus",
  "qwen.enableThinking": true,
  "qwen.enableFast": false,
  "qwen.enableAuto": true,
  "qwen.partialMode": true,
  "qwen.acceptMode": "tab"
}
```

---

## 💡 РЕЖИМЫ РАБОТЫ

### 1. Режим принятия правок (Tab)

**Когда использовать:**
- ✅ Повседневная разработка
- ✅ Рефакторинг
- ✅ Исправление багов

**Как работает:**
```
ИИ предлагает изменения → Вы нажимаете Tab → Изменения применяются
```

---

### 2. Режим YOLO

**Когда использовать:**
- ✅ Массовое форматирование
- ✅ Простые изменения
- ✅ Переименование

**Предупреждение:**
> ⚠️ Изменения применяются автоматически! Проверяйте код после применения.

---

### 3. Режим планирования

**Когда использовать:**
- ✅ Сложные многошаговые задачи
- ✅ Архитектурные изменения
- ✅ Новые функции

**Как работает:**
```
Запрос → План → Проверка → Выполнение
```

---

### 4. Обычный режим (фиолетовый >)

**Когда использовать:**
- ✅ Вопросы
- ✅ Короткие команды
- ✅ Объяснения кода
