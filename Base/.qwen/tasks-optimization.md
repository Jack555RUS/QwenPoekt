## 🔴 ЗАДАЧА 1: Оптимизация расходов (экономия токенов)

**Описание:**
Увеличение количества обработок за один раз за счёт экономии токенов.

**Проблема:**
- ИИ тратит токены на изучение структуры проекта каждый раз
- Контекст не сохраняется между сессиями
- Повторение одной и той же информации

**Решение:**
1. ✅ **CLAUDE.md** — главный контекст (экономия на объяснениях)
2. ✅ **Атомарные знания** — 20-30 строк вместо 500
3. ✅ **Иерархия контекста** — корень + поддиректории
4. ⏳ **Сжатие контекста** — удалять дубликаты из переписки
5. ⏳ **Кэширование** — сохранять часто используемые промпты

**Ожидаемый результат:**
- ✅ Экономия 30-50% токенов на сессию
- ✅ Увеличение обработок за раз в 2x
- ✅ Быстрое восстановление контекста

**Статус:** ⏳ В работе (CLAUDE.md создан)

## 🔴 ЗАДАЧА 2: Полная копия Base (с хаосом)

**Описание:**
Создать полную копию Base перед наведением порядка.

**План:**
```powershell
# 1. Создать копию со всем содержимым
xcopy "D:\QwenPoekt\Base" "D:\QwenPoekt\_BACKUP\Base_Before_Cleanup" /E /I /H /Y

# 2. Проверить целостность
dir "D:\QwenPoekt\_BACKUP\Base_Before_Cleanup" /s
```

**Что копируем:**
- ✅ ВСЁ содержимое Base/
- ✅ Включая OLD/, BOOK/, KNOWLEDGE_BASE/
- ✅ Включая _TEMP/, *.log, *.tmp
- ✅ Включая sessions/

**Зачем:**
- ✅ Страховка перед изменениями
- ✅ Можно откатиться к любой точке
- ✅ Тестирование на копии

**Статус:** ⏳ Ожидает выполнения

## 🟡 ЗАДАЧА 3: Разделение Base и проектов

**Описание:**
База отдельно, проекты отдельно, библиотека едина.

**Структура:**
```
D:\QwenPoekt\
├── Base/                      # ✅ БАЗА (инструменты ИИ)
│   ├── 01-Projects/          # Текущие задачи
│   ├── 02-Areas/             # Стандарты
│   ├── 03-Resources/         # Ресурсы
│   ├── 04-Archives/          # Архив
│   ├── sessions/             # Сессии ИИ
│   └── _BACKUP/              # Бэкапы
│
├── PROJECTS/                  # ✅ ПРОЕКТЫ (отдельно)
│   ├── DragRaceUnity/
│   ├── OtherProject/
│   └── ...
│
└── KNOWLEDGE_LIBRARY/         # ✅ БИБЛИОТЕКА ЗНАНИЙ (едина)
    ├── Books/                # Книги
    ├── Documentation/        # Документация
    ├── Tutorials/            # Обучение
    └── References/           # Справочники
```

**Принципы:**
- **Base:** Инструменты для работы ИИ (скрипты, конфиги, правила)
- **PROJECTS:** Конкретные проекты (DragRaceUnity, etc.)
- **KNOWLEDGE_LIBRARY:** Единая библиотека знаний (книги, документация, инструкции)
