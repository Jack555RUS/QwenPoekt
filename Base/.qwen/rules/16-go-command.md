---

## 🎯 НАЗНАЧЕНИЕ

Команда `/go` предназначена для **автоматической загрузки всей базы знаний** из `000-BASE_INDEX.md`.

**Главный принцип:** Одна команда → Полная загрузка → Готов к работе

---

## 📋 КОГДА ИСПОЛЬЗОВАТЬ

| Ситуация | Использовать /go |
|----------|------------------|
| **Старт новой сессии** | ✅ Да |
| **Потеря контекста** | ✅ Да |
| **Перед сложной задачей** | ✅ Да |
| **После перезапуска Qwen Code** | ✅ Да |
| **Нужно "полное погружение"** | ✅ Да |
| **Работа с неизвестной темой** | ✅ Да |

---

## 🚀 ИСПОЛЬЗОВАНИЕ

### Базовое использование:

```
/go
```

### Что делает ИИ:

```
1. Читает 000-BASE_INDEX.md
2. Извлекает список файлов (7 уровней, 46 файлов)
3. Читает каждый файл через инструменты Qwen Code
4. Докладывает о загрузке
```

## 💡 АЛЬТЕРНАТИВЫ

| Команда | Когда использовать | Отличие от /go |
|---------|-------------------|----------------|
| **`/lazy-load [тема]`** | Нужна только одна тема | Загружает 3-5 файлов вместо 46 |
| **`/resume`** | Восстановление сессии | Загружает контекст сессии, не всю базу |
| **`/architect-mode`** | Нужен полный анализ задачи | Включает анализ, не загрузку файлов |

| Ограничение | Описание |

| **Лимит токенов** | Загрузка занимает несколько сообщений |
| **Не автоматическая** | Нужно явно написать `/go` в чат |

## 🎯 ИСТОРИЯ СОЗДАНИЯ

**Дата:** 5 марта 2026 г.

**Причина:**
- Пользователь устал объяснять каждый раз что загружать
- Нужна команда "полной загрузки" одним словом
- Долгий путь анализа (17 правил, 46 файлов) привёл к этому решению

**Решение:**
- Команда `/go` загружает всё автоматически
- Правило зафиксировано для будущего использования
