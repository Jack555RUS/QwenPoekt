# 15 — BASE INDEX AUTO-UPDATE (АВТОМАТИЧЕСКОЕ ОБНОВЛЕНИЕ)

**Версия:** 1.0
**Дата:** 4 марта 2026 г.
**Статус:** 🔴 **КРИТИЧНО**
**Модуль:** `.qwen/rules/15-base-index-auto-update.md`

---

## 🎯 НАЗНАЧЕНИЕ

Это правило обеспечивает **автоматическое обновление** `000-BASE_INDEX.md` при изменениях в папке `.qwen/rules/`.

**Главный принцип:** Одно изменение → Одно обновление BASE_INDEX

---

## 📋 ОБЛАСТЬ ПРИМЕНЕНИЯ

**Когда применять (автоматически):**

| Событие | Действие | Триггеры |
|---------|----------|----------|
| **Создано правило** | Добавить в BASE_INDEX | `*.md` в `.qwen/rules/` |
| **Удалено правило** | Удалить из BASE_INDEX | Файл удалён из `.qwen/rules/` |
| **Переименовано правило** | Обновить ссылку | Старое имя → Новое имя |
| **Перемещено правило** | Обновить путь | Изменён путь файла |

**Когда НЕ применять:**

| Ситуация | Причина |
|----------|---------|
| Изменение содержимого правила | Не влияет на список |
| Временные файлы (*.tmp) | Не правила |
| Тестовые файлы (99-*, test-*) | Не критичные |

### ШАГ 1: ОБНАРУЖЕНИЕ ИЗМЕНЕНИЯ

**Мониторинг:**
```powershell
# Проверка изменений в папке правил
Get-ChildItem .qwen/rules/ -Filter "*.md" | 
    Where-Object { $_.LastWriteTime -gt (Get-Date).AddMinutes(-5) }
```

**Триггеры:**
- ✅ Создан новый файл `.md`
- ✅ Удалён файл `.md`
- ✅ Переименован файл `.md`

## ⚠️ ПОДВОДНЫЕ КАМНИ

| Проблема | Решение |
|----------|---------|
| **BASE_INDEX не найден** | Проверить путь: `000-BASE_INDEX.md` в корне Base/ |
| **Ссылка не работает** | Проверить путь: `.qwen/rules/XX-filename.md` |
| **Счётчик не сходится** | Пересчитать: `Get-ChildItem .qwen/rules/*.md \| Measure-Object` |
| **Конфликт версий** | Всегда увеличивать версию: 1.0 → 1.1 → 1.2 |
| **Забыли коммит** | Проверить: `git status` после обновления |
