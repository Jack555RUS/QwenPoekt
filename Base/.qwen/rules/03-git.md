## 📋 ПРАВИЛО PUSH

| Ситуация | Действие |
|----------|----------|
| **Ежедневно** | Коммит локально (auto-commit-daily.ps1 в 18:00) |
| **Крупные изменения** | PUSH сразу (завершил задачу, исправил ошибку) |
| **По запросу** | `/backup` — коммит + push |
| **Не нужно** | PUSH каждый день (ждём крупного изменения) |

**Проверка:**
```powershell
git status
```
- `ahead by X commits` → Можно push (если крупное изменение)
- `up to date` → PUSH не нужен

## 📂 OLD ПАПКА (БИБЛИОТЕКА НАРАБОТОК)

**OLD = Не мусор, а библиотека незавершённых проектов!**

### Структура:

```
OLD/
├── _INBOX/               # Новые (7 дней, анализ)
├── _ANALYZED/            # Проанализированы (60 дней)
├── _IDEAS/               # Идеи (вечно) ⭐⭐⭐⭐⭐
├── _CODE_SNIPPETS/       # Готовый код (вечно) ⭐⭐⭐⭐⭐
└── _ARCHIVE_60D/         # На удаление (>60 дней)
```

### Сроки хранения:

| Папка | Срок | Что делать |
|-------|------|------------|
| `OLD/_INBOX/` | 7 дней | Анализ (`old-analysis.ps1`) |
| `OLD/_ANALYZED/` | 60 дней | Ждать решения |
| `OLD/_ARCHIVE_60D/` | 60 дней | Удалить (после анализа) |
| `OLD/_IDEAS/` | ✅ Вечно | Использовать в проектах |
| `OLD/_CODE_SNIPPETS/` | ✅ Вечно | Библиотека кода |

### Команды:

```powershell
# Переместить заброшенный проект в OLD
.\scripts\move-to-old.ps1 -ProjectPath "PROJECTS/OldProject" -Reason "Abandoned"

# Анализ OLD (извлечь идеи, код)
.\scripts\old-analysis.ps1

# Очистка OLD/_ARCHIVE_60D/ (>60 дней)
.\scripts\old-cleanup.ps1                # Отчёт
.\scripts\old-cleanup.ps1 -AutoConfirm   # Удаление
```

## 📦 RELEASE ПАПКА (ГОТОВЫЕ ПРОЕКТЫ)

**Назначение:** Готовые, рабочие проекты (только доработки)

### Структура:

```
RELEASE/
├── ProjectName_v1.0/      # Версия 1.0
│   ├── Build/             # EXE файл
│   ├── Source/            # Исходники (важное)
│   ├── Documentation/     # Документация
│   └── README.md          # Инструкция
└── Template_MainMenu/     # Шаблоны
```

### Правила:

- ✅ Хранить только готовые, протестированные версии
- 📦 Версионирование: `v1.0`, `v1.1`, `v2.0`
- 📄 Обязательно: README.md
- ✅ Тесты: Обязательно (если код)
- 🔧 Изменения: Только доработки (новые версии)
