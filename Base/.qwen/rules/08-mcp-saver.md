Это правило описывает использование **MCP Session Saver** для прямого сохранения сессий.

- `mcp.json` — подключение к Qwen Code

---

## 🛠️ ИНСТРУМЕНТЫ MCP

### 1. save_session

**Сохранить сессию.**

**Параметры:**
- `chat` (строка) — диалог (Markdown)
- `sessionId` (строка) — ID сессии
- `metadata` (объект, опционально) — метаданные
- `task` (строка, опционально) — задача

**Пример:**
```
save_session({
  "chat": "# Диалог...",
  "sessionId": "2026-03-03_14-30",
  "task": "Настройка MCP"
})
```

---

### 2. load_session

**Загрузить сессию.**

**Параметры:**
- `sessionId` (строка) — ID сессии

---

### 3. list_sessions

**Список сессий.**

**Параметры:**
- `limit` (число, опционально) — максимум сессий

---

### 4. update_resume_marker

**Обновить маркер восстановления.**

**Параметры:**
- `sessionId` (строка) — ID сессии
- `currentTask` (строка, опционально) — задача
- `nextStep` (строка, опционально) — шаг

---

## 🔄 ПОВЕДЕНИЕ ИИ

### При каждом ответе:

1. Сформировать диалог
2. Вызвать `save_session()`
3. Получить подтверждение
