# 07 — СОХРАНЕНИЕ СЕССИЙ (SESSION PERSISTENCE)

**Версия:** 1.0
**Дата:** 2 марта 2026 г.
**Статус:** ✅ Активно
**Модуль:** `.qwen/rules/07-session-persistence.md`

---

## 🎯 НАЗНАЧЕНИЕ

Это правило определяет поведение ИИ при:
1. ✅ **Автосохранении переписки** (каждые 2 минуты)
2. ✅ **Восстановлении контекста** (после перезагрузки)
3. ✅ **Обработке триггеров** (выход, перезагрузка)
4. ✅ **Логировании команд** (PowerShell transcript)

---

## 📋 КОНФИГУРАЦИЯ

**Файл:** `.qwen/session-rules.json`

**Основные настройки:**
```json
{
  "session": {
    "auto_save": {
      "enabled": true,
      "interval_minutes": 2,
      "format": "hybrid",
      "path": "sessions/"
    },
    "auto_resume": {
      "enabled": true,
      "check_on_start": true
    },
    "chat_log": {
      "enabled": true,
      "include_commands": true
    }
  }
}
```

---

## 🔄 ПОВЕДЕНИЕ ИИ

### При старте сессии:

**1. Прочитать конфигурацию:**
```powershell
.\scripts\check-session-rules.ps1
```

**2. Проверить:**
- ✅ `.qwen/session-rules.json` существует?
- ✅ `auto_save.enabled = true`?
- ✅ Задача `QwenSessionAutoSave` существует?
- ✅ `.resume_marker.json` существует? (восстановление)

**3. Если найдена предыдущая сессия:**
```
🔄 Найдена предыдущая сессия!

📊 Информация:
  Дата: 2026-03-02 22:00
  Задача: Установка RAM диска
  Последнее сохранение: 2 мин назад

💡 Используйте /resume для восстановления
```

---

### При триггерах (выход/перезагрузка):

**1. Обнаружить триггер:**
```
Слова: "выход", "перезагрузка", "выключение", "stop", "break"
```

**2. Проверить конфигурацию:**
```powershell
$rules = Get-Content ".qwen/session-rules.json" | ConvertFrom-Json
if ($rules.session.auto_save.enabled) {
    # Сохранить сессию
}
```

**3. Сохранить переписку:**
```
💾 Сохраняю сессию...

✅ chat.md (245 KB)
✅ metadata.json (3 KB)
✅ commands.log (12 KB)

📁 sessions/2026-03-02_22-00/
```

**4. Предложить восстановление:**
```
После перезагрузки используйте /resume для восстановления контекста.
```

---

### При команде /save-session:

**1. Сформировать переписку:**
```markdown
# Сессия: 2026-03-02 22:00
**Задача:** Установка RAM диска

---

## [22:00:00] Пользователь
Привет! Нужно установить RAM диск.

## [22:00:15] ИИ
Отлично! Начинаю настройку...
```

**2. Сформировать метаданные:**
```json
{
  "session_id": "2026-03-02_22-00",
  "timestamp": "2026-03-02T22:00:00",
  "message_count": 45,
  "command_count": 12
}
```

**3. Сохранить:**
```powershell
.\scripts\save-chat-log.ps1 -ChatContent $chat -Metadata $meta
```

---

### При автосохранении (каждые 2 мин):

**1. Task Scheduler запускает:**
```powershell
.\scripts\auto-save-chat.ps1
```

**2. Скрипт запрашивает у ИИ:**
- Текущую переписку
- Метаданные сессии
- Список команд

**3. Сохраняет:**
```
sessions/2026-03-02_22-04/
├── chat.md (100 KB)
├── metadata.json (1 KB)
└── commands.log (5 KB)
```
