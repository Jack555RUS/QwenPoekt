**НОВОЕ:** Перед любым действием применить [`10-think-before-action.md`](./10-think-before-action.md)

### 🔴 КРАСНЫЙ — ВСЕГДА СПРАШИВАТЬ

**Когда:**
- Удаление файлов/папок (>10 файлов)
- Перемещение, переименование файлов
- Выбор между вариантами решения
- **Создание новых скриптов/правил/документации**
- Изменение правил иерархии
- Интеграция новых знаний

**Формат вопроса:**
```
Какой вариант предпочитаете?
- 1️⃣ Вариант 1 — [кратко]
- 2️⃣ Вариант 2 — [кратко]
- 💡 Свой вариант
```

---

### 🟡 ЖЁЛТЫЙ — ПО КОНТЕКСТУ

**Когда:**
- Обновление документации
- Актуализация правил
- Добавление ссылок/связей
- Проверка правил

**Контекстные триггеры:**
- **"сделай", "выполни", "обнови"** → Действовать автономно
- **"предложи", "как лучше", "что думаешь"** → Спросить выбор
- **Сомневаешься** → Спросить

---

### 🟢 ЗЕЛЁНЫЙ — АВТОНОМНО

**Когда:**
- Коммиты, тесты, проверки
- Копирование файлов (сохранение)
- Очистка временных файлов
- Исправление ошибок в скриптах
- Стандартные операции (по правилам)

**Условия:**
- ✅ Есть явное разрешение Пользователя
- ✅ Действие по известному правилу
- ✅ Не влияет на структуру проекта
