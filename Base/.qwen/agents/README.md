**Важно:** Агенты подчиняются МНЕ, а не Пользователю!

---

## 📋 МОИ АГЕНТЫ

### 1. 🎓 Learning Agent (Автономное обучение)

**Роль:** Автономное пополнение базы знаний

**Режим:** Фоновый (не мешает работе)

**Задача от БОССА:** Изучать новые ресурсы, систематизировать, сохранять в базу

**Отчётность:** Еженедельный отчёт БОССУ (Понедельник 09:00)

**Команда БОССА:**
```
/learning-agent [команда]
```

**Файл:** `learning-agent.md`

---

### 2. 🛡️ Code Reviewer

**Роль:** Проверка кода на соответствие стандартам

**Команда:**
```
/code-reviewer проверь [файл/модуль]
```

**Специализация:**
- ✅ Стиль кода (.editorconfig)
- ✅ Уязвимости
- ✅ Производительность
- ✅ Конституция AI

**Файл:** `code-reviewer.md`

---

### 2. 🧪 Test Writer

**Роль:** Написание юнит-тестов

**Команда:**
```
/test-writer напиши тесты для [модуль]
```

**Специализация:**
- ✅ Юнит-тесты (NUnit, UTF)
- ✅ Интеграционные тесты
- ✅ Play Mode тесты
- ✅ Edit Mode тесты

**Файл:** `test-writer.md`

---

### 3. 🔒 Security Analyzer

**Роль:** Анализ безопасности

**Команда:**
```
/security-analyzer проверь [файл/модуль]
```

**Специализация:**
- ✅ Поиск уязвимостей
- ✅ Безопасность данных
- ✅ Права доступа
- ✅ Инъекции

**Файл:** `security-analyzer.md`

---

### 4. 📚 Documentation Writer

**Роль:** Создание документации

**Команда:**

/doc-writer создай документацию для [модуль]
```

**Специализация:**
- ✅ XML документация

- ✅ API документация

**Файл:** `documentation-writer.md`

### 5. 🚀 Build Master

**Роль:** Автоматизация сборки

**Команда:**
```

```

**Специализация:**
- ✅ Автосборка

- ✅ CI/CD пайплайны

**Файл:** `build-master.md`

## 🎯 КАК ИСПОЛЬЗОВАТЬ

### Синтаксис:

```

```bash

# Документация

# Сборка
/build-master собери проект
```

## 🔄 ЦИКЛ РАБОТЫ С АГЕНТАМИ

```
1. Пользователь выбирает агента
2. Запрашивает задачу
3. Агент выполняет
4. Предоставляет отчёт
5. Пользователь проверяет
6. При необходимости → исправления
```
