## 📁 ОПИСАНИЕ ПАПОК

### Base/ — База Знаний

**Назначение:** Хранение всех знаний, скриптов, документации

**Размер:** ~52 MB (80 000+ файлов)

**Ключевые файлы:**

| Файл | Назначение |
|------|------------|
| `AI_START_HERE.md` | Главная инструкция проекта |
| `README.md` | Навигатор по проекту |
| `.qwen/QWEN.md` | Мастер-конфиг ИИ |
| `RULES_AND_TASKS.md` | Правила и задачи |
| `NOTES.md` | Заметки сессии |
| `ТЕКУЩАЯ_ЗАДАЧА.md` | Текущая задача |

**Правила:**
- ✅ Хранить все знания здесь
- ✅ Использовать snake_case для имён
- ✅ Вести OPERATION_LOG.md
- ✅ Git коммит важных изменений

## 📋 ПРАВИЛА ИМЕНОВАНИЯ

### Файлы

**Правила:**

- ✅ **Латиница** (a-z, A-Z)
- ✅ **Нижний регистр** (file_name.md)
- ✅ **Подчёркивание** (не пробелы)
- ✅ **Даты:** ГГГГ-ММ-ДД (2026-03-02)

**Примеры:**
- ✅ `file_naming_rule.md`
- ✅ `pre-operation-backup.ps1`
- ✅ `2026-03-02_analysis.md`
- ❌ `File Naming Rule.md`
- ❌ `файл.md`

---

### Папки

**Правила:**

- ✅ **Латиница** (A-Z, a-z)
- ✅ **PascalCase** (MyFolder)
- ✅ **Подчёркивание** (My_Folder)

**Примеры:**
- ✅ `KNOWLEDGE_BASE`
- ✅ `_BACKUP`
- ✅ `00_CORE`
- ❌ `База Знаний`
