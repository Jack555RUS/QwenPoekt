## 🚀 Как запустить

### Через Unity Hub
1. Открыть Unity Hub
2. Добавить проект: `D:\QwenPoekt\ProbMenu\DragRaceUnity`
3. Открыть сцену `MainMenu`
4. Нажать Play

### Сборка
1. File → Build Settings
2. Добавить сцены: MainMenu, GameMenu, Race, Garage, Tuning, Shop
3. Build
