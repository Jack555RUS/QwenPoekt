# Pre-Operation Backup

**Назначение:** Автоматическое создание резервной копии ПЕРЕД удалением/перемещением файлов

---

## 🚀 БЫСТРЫЙ СТАРТ

### Перед операцией (удаление/перемещение):

```powershell
# 1. Проверка (без создания бэкапа)
.\scripts\pre-operation-backup.ps1 -OperationType "Delete_OldProject" -WhatIf

# 2. Создание бэкапа (обязательно!)
.\scripts\pre-operation-backup.ps1 -OperationType "Delete_OldProject"

# 3. Теперь можно безопасно выполнять операцию
Remove-Item "OLD/_INBOX/AbandonedProject" -Recurse -Force
```

---

## 📋 ПАРАМЕТРЫ

| Параметр | Обязательный | Описание | Пример |
|----------|--------------|----------|--------|
| `-OperationType` | ✅ Да | Тип операции (для имени папки) | `"Delete_OldProject"` |
| `-BackupRoot` | ❌ Нет | Где хранить бэкапы | `"D:\QwenPoekt\_BACKUP"` |
| `-SourcePath` | ❌ Нет | Что копировать | `"D:\QwenPoekt\Base"` |
| `-WhatIf` | ❌ Нет | Только проверка (без бэкапа) | `-WhatIf` |

---

## 📖 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Перед удалением папки OLD

```powershell
# Шаг 1: Создать бэкап
.\scripts\pre-operation-backup.ps1 -OperationType "Delete_AbandonedProject"

# Шаг 2: Удалить папку
Remove-Item "OLD/_INBOX/AbandonedProject" -Recurse -Force

# Шаг 3: Записать в OPERATION_LOG.md
```

### Пример 2: Перед перемещением скриптов

```powershell
# Шаг 1: Создать бэкап
.\scripts\pre-operation-backup.ps1 -OperationType "Move_Scripts"

# Шаг 2: Переместить файлы
Move-Item "03-Resources/PowerShell/old/*.ps1" "OLD/_ARCHIVE/" -Force

# Шаг 3: Записать в OPERATION_LOG.md
```

### Пример 3: Проверка (без бэкапа)

```powershell
# Узнать размер и количество файлов
.\scripts\pre-operation-backup.ps1 -OperationType "Test" -WhatIf

# Вывод:
# Файлов: 82 735
# Размер: 2 508.62 MB
# ⚠️ РЕЖИМ ПРОВЕРКИ (WhatIf)
```

## ⚠️ ВАЖНЫЕ ПРАВИЛА

1. **ВСЕГДА создавай бэкап ПЕРЕД удалением/перемещением**
2. **Проверяй целостность** (скрипт делает это автоматически)
3. **Записывай в OPERATION_LOG.md** (скрипт делает это автоматически)
4. **Храни 45 дней** (потом глубокий анализ перед удалением)
5. **_BACKUP вне Base** (чтобы при сбое Base бэкап не потерялся)
