## 🚀 ИСПОЛЬЗОВАНИЕ

**Скажите мне:**

```

```

**Я автоматически:**

1. ✅ Запущу `.\scripts\end-session.ps1`
2. ✅ Создам Git коммит
3. ✅ Обновлю OPERATION_LOG.md
4. ✅ Создам отчёт

---

### Вариант 2: Ручной запуск

```powershell

.\scripts\end-session.ps1
```

---

## 📊 ЧТО ДЕЛАЕТ СКРИПТ

```
1. Проверка изменений (Git status)
   ↓
2. Git add + commit (если есть изменения)
   ↓
3. Обновление OPERATION_LOG.md
   ↓
4. Создание отчёта (reports/SESSION_ГГГГ-ММ-ДД_ЧЧ-мм.md)
   ↓
5. Вывод статистики
```
