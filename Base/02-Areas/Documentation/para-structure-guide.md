**Критерий:** Есть дедлайн или цель завершения

### **02-Areas (Области ответственности)**

**Критерий:** Постоянная ответственность (без дедлайна)

| Область | Описание | Стандарт |
|---------|----------|----------|
| **Code-Quality** | Качество кода | StyleCop, SonarLint |
| **Documentation** | Документация | Markdown, актуальность |
| **Session-Management** | Сессии ИИ | Автосохранение 2 мин |
| **Backup-System** | Бэкапы | 3 уровня защиты |

**Правило:** Поддерживать стандарт, обновлять еженедельно

## ⚠️ ТИПИЧНЫЕ ОШИБКИ

| Ошибка | Решение |
|--------|---------|
| Создание папок по темам | Организовать по действиям |
| Глубокая вложенность (3+ уровня) | Максимум 2 уровня |
| Смешивание Projects и Areas | Разделять по дедлайну |
| Попытка организовать всё старое | Архивировать, начать с чистого листа |
| Папка «Разное» | Не допускать исключений |
