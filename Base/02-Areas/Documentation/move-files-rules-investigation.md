**Проблема:**
```

```

**Последствие:**

**Решение:**
```powershell

param($Source, $Destination, $Filter)

# 4. Отчёт о влиянии
```

---

### **2. Нет правила "Массовое перемещение"**

**Проблема:**
```
Правила есть для единичных операций
Нет правил для массовых операций (>100 файлов)
```

**Последствие:**
```
Массовое перемещение выполнено без подтверждения
```

**Решение:**
```markdown
## 🟣 ФИОЛЕТОВЫЙ уровень — МАССОВЫЕ ОПЕРАЦИИ

**Когда:**
- Перемещение >100 файлов
- Переименование >50 файлов
- Реорганизация структуры

**Действие:**
1. Создать план перемещения
2. Запустить pre-move-check.ps1
3. Получить подтверждение пользователя
4. Выполнить перемещение
5. Проверить ссылки
6. Закоммитить
```

# 2. Заменить ссылки
foreach ($file in $mdFiles) {

    $content = $content -replace $OldPath, $NewPath
    $content | Out-File $file -Encoding utf8
}
