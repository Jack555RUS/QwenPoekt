## 🚀 Быстрый старт

### 1. Открыть проект
```
1. Unity Hub → Add → Add project from disk
2. Путь: D:\QwenPoekt\PROJECTS\DragRaceUnity
3. Открыть проект
```

### 2. Запустить в Editor
```
1. Открыть: Assets/Scenes/MainMenu.unity
2. Нажать Play
3. Проверить кнопки
```

### 3. Создать EXE
```powershell
cd D:\QwenPoekt\PROJECTS\DragRaceUnity
.\auto-build-exe.ps1
```
