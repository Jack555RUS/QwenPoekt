# ⚡ Быстрая установка SonarLint

**Время установки:** 2 минуты
**Лицензия:** Бесплатно (open source)

**Другие бесплатные инструменты:** см. **[FREE_ANALYSIS_TOOLS.md](./FREE_ANALYSIS_TOOLS.md)**

---

## Шаг 1: Установка расширения

```
1. Откройте Visual Studio
2. Extensions → Manage Extensions (Расширения → Управление расширениями)
3. В поиске введите: SonarLint
4. Нажмите Download
5. Закройте Visual Studio для установки
6. Дождитесь установки
7. Снова откройте Visual Studio
```

---

## Шаг 2: Первая настройка

```
1. Tools → Options → SonarLint
2. Включите опции:
   ☑ Activate on solution open
   ☑ Analysis on the fly
3. Нажмите OK
```

---

## Шаг 3: Проверка работы

```
1. Откройте проект DragRaceUnity.sln
2. Откройте любой .cs файл
3. SonarLint автоматически начнёт анализ
4. Проблемы подсветятся в редакторе
```

---

## 🔍 Как использовать

### В редакторе кода:
- **Красное подчёркивание** = Ошибка
- **Жёлтое подчёркивание** = Предупреждение
- **Синее подчёркивание** = Рекомендация

### Быстрое исправление:
```
1. Кликните на подчёркивание
2. Нажмите Ctrl+. (Control + точка)
3. Выберите исправление из списка
```

### Окно ошибок:
```
View → Error List (Вид → Список ошибок)
Фильтр: Show → Errors + Warnings + Messages
```

---

## 🎯 Что найдёт SonarLint в проекте

### Примеры проблем:
- ❌ Неиспользуемые переменные
- ❌ Нарушения стиля кода
- ❌ Потенциальные null reference exceptions
- ❌ Code smells (запахи кода)
- ❌ Уязвимости безопасности
