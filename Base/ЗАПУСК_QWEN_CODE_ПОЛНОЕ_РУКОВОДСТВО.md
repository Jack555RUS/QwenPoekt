## 1. БЫСТРЫЙ СТАРТ

### 1.1: Обычный запуск

**Шаги:**
```
1. Открыть VS Code
2. Открыть папку: D:\QwenPoekt\Base
3. Открыть терминал (Ctrl+`)
4. Написать: qwen
5. Нажать Enter
```

**Результат:**
```
✅ Qwen Code запускается
✅ Загружается история чатов
✅ Готов к работе
```

---

### 1.2: Восстановление сессии

**Шаги:**
```
1. Открыть терминал (Ctrl+`)
2. Запустить: .\resume-session.ps1
3. Скопировать контекст (автоматически)
4. Перейти в чат Qwen Code
5. Вставить (Ctrl+V)

7. Нажать Enter
```

**Результат:**
```
✅ Контекст загружен
✅ ИИ "вспомнил" сессию
✅ Задачи восстановлены
```

## 2. ПОДРОБНОЕ РУКОВОДСТВО

### 2.1: Установка и настройка

**Требования:**
- ✅ Node.js 20+
- ✅ VS Code
- ✅ Qwen Code extension

**Проверка:**
```powershell
node -v
# Должно быть: v20.x.x или выше
```

**Установка Qwen Code:**
```powershell
npm install -g @qwen-code/qwen-code@latest
```

**Аутентификация:**
```
1. Запустить: qwen
2. В чате: /auth
3. Выбрать: Qwen OAuth
4. Войти в браузере
5. Разрешить доступ
```

---

### 2.2: Структура проекта


### 2.3: Команды Qwen Code

**Основные:**
| Команда | Описание |
|---------|----------|
| `qwen` | Запуск Qwen Code |
| `/auth` | Аутентификация |
| `/model` | Смена модели |
| `/resume` | История чатов (не контекст!) |
| `/save-session` | Сохранение сессии |
| `/insight` | Персонализированные выводы |

**Наши команды:**
| Команда | Описание |
|---------|----------|
| `.\resume-session.ps1` | Восстановление контекста |
| `/save-session` | Сохранить контекст в .resume_marker.json |

```
   ↓
```

---

### 2.6: Сохранение сессии

**Перед завершением:**
```
1. В чате: /save-session
   ↓
2. Контекст сохраняется в .resume_marker.json
   ↓
3. Сессия сохраняется в sessions/YYYY-MM-DD_HH-mm/
   ↓
4. Можно закрыть VS Code
```

**Автоматическое сохранение:**
```
✅ Каждые 2 минуты (Task Scheduler)
✅ При закрытии VS Code
✅ При команде /save-session
```

---

### 2.7: Устранение неполадок

**Проблема 1: qwen не найден**
```
Решение:
npm install -g @qwen-code/qwen-code@latest
```

**Проблема 2: .resume_marker.json не найден**
```
Решение:
1. Запустить: qwen
2. В чате: /save-session
3. Контекст сохранится
```

**Проблема 3: resume-session.ps1 не работает**
```
Решение:
powershell -ExecutionPolicy Bypass -File ".\resume-session.ps1"
```

**Проблема 4: ИИ не "вспомнил" контекст**
```
Решение:
1. Убедиться что .resume_marker.json существует
2. Вставить полный контекст в чат
3. Написать: "Восстанови контекст сессии"
```
