## 📋 ПРАВИЛО 1: ПРОВЕРКА ССЫЛОК ПЕРЕД ПЕРЕМЕЩЕНИЕМ

**Проблема:**
- Переместил файлы → ссылки сломались
- Пришлось исправлять вручную (9 ссылок в отчётах)

**Решение:**

### Перед перемещением:

```powershell
# 1. Найти все ссылки на файл
grep_search -pattern "CHEAT_SHEET.md" -glob "*.md"

# 2. Посчитать количество
# 3. Оценить масштаб изменений
```

### Чек-лист перед перемещением:

```
[ ] 1. НАЙТИ ВСЕ ССЫЛКИ
    [ ] grep_search по всем .md файлам
    [ ] Посчитать количество ссылок
    
[ ] 2. ОЦЕНИТЬ МАСШТАБ
    [ ] < 10 ссылок → исправить вручную
    [ ] 10-50 ссылок → создать скрипт
    [ ] > 50 ссылок → отложить или автоматизировать
    
[ ] 3. СОЗДАТЬ СКРИПТ ИСПРАВЛЕНИЯ
    [ ] edit для каждого файла
    [ ] Или replace в текстовом редакторе
    
[ ] 4. ИСПРАВИТЬ ССЫЛКИ
    [ ] ./file.md → ../02-Areas/Documentation/file.md
    [ ] Проверить относительные пути
    
[ ] 5. ПРОВЕРИТЬ
    [ ] grep_search ещё раз (должно быть 0)
    [ ] Git status (только изменения ссылок)
    
[ ] 6. ЗАКОММИТИТЬ
    [ ] Отдельный коммит: "Fix: ссылки после перемещения"
```

### Автоматизация (скрипт):

```powershell
# scripts\fix-links-after-move.ps1

param(
    [string]$OldPath = "./CHEAT_SHEET.md",
    [string]$NewPath = "../02-Areas/Documentation/CHEAT_SHEET.md",
    [string]$SearchPath = "."
)

# Найти все файлы со ссылками
$files = Get-ChildItem -Recurse -Filter "*.md" | 
    Select-String -Pattern $OldPath |
    Select-Object -ExpandProperty Path -Unique

foreach ($file in $files) {
    $content = Get-Content $file -Raw
    $content = $content -replace [regex]::Escape($OldPath), $NewPath
    $content | Out-File $file -Encoding UTF8
    Write-Host "✓ Исправлено: $file"
}
```

**Проблема:**
- Забыл записать задачу → не выполнил
- Записал в случайное место → не нашёл

- [ ] Задача 2

**Срок:** 2026-03-01

**Зависимости:**
- [x] Организация корня (завершено)

- [ ] Исправить ссылки в KNOWLEDGE_BASE/
- [ ] Исправить ссылки в reports/

**Результат:**
[Заполняется после выполнения]

## 📋 ПРАВИЛО 4: АВТОМАТИЗАЦИЯ ПРОВЕРОК

**Проблема:**
- Забыл проверить → ошибка
- Забыл записать → потерял

**Решение:**

### Скрипт проверки перед коммитом:

```powershell
# scripts\pre-commit-check.ps1

Write-Host "=== PRE-COMMIT CHECK ==="

# 1. Проверить ссылки на несуществующие файлы
Write-Host "1. Проверка ссылок..."
$brokenLinks = Get-ChildItem -Recurse -Filter "*.md" |
    Select-String -pattern "\[.*?\]\(.*?\)" |
    Where-Object { 
        $path = $_.Matches.Value -replace "\[(.*?)\]\((.*?)\)", '$2'
        !(Test-Path $path)
    }

if ($brokenLinks) {
    Write-Host "❌ Найдены битые ссылки:" -ForegroundColor Red
    $brokenLinks | ForEach-Object { Write-Host "  - $($_.Path)" }
    exit 1
}

# 2. Проверить NOTES.md на незавершённые задачи
Write-Host "2. Проверка задач..."
$tasks = Get-Content "NOTES.md" | Select-String "⏳"
if ($tasks) {
    Write-Host "⚠️ Есть незавершённые задачи:" -ForegroundColor Yellow
    $tasks | ForEach-Object { Write-Host "  - $($_.Line)" }
}

# 3. Проверить .gitignore
Write-Host "3. Проверка .gitignore..."
if (!(Test-Path ".gitignore")) {
    Write-Host "❌ .gitignore не найден!" -ForegroundColor Red
    exit 1
}

Write-Host "✅ Pre-commit check passed!" -ForegroundColor Green
```
