## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Конфигурация:

```json
{
  "server": "session-saver",
  "sdk": "@modelcontextprotocol/sdk@1.27.1",
  "class": "McpServer",
  "method": "registerTool()",
  "transport": "StdioServerTransport"
}
```

### Инструменты MCP:

1. **list_sessions** — список сессий
   - Вход: `limit` (number, optional)
   - Выход: список сессий с метаданными

2. **save_session** — сохранение сессии
   - Вход: `chat`, `sessionId`, `metadata`, `task`
   - Выход: подтверждение сохранения

3. **load_session** — загрузка сессии
   - Вход: `sessionId`
   - Выход: chat.md + metadata.json

4. **update_resume_marker** — обновление маркера
   - Вход: `sessionId`, `currentTask`, `nextStep`
   - Выход: подтверждение обновления

## 🐛 ПРОБЛЕМЫ И РЕШЕНИЯ

### Проблема 1: SDK 0.5.0 не работает

**Ошибка:**
```
TypeError: Cannot read properties of undefined (reading 'method')
```

**Причина:** Неправильный API для версии SDK

**Решение:**
- Обновлено до SDK 1.27.1
- Использован `McpServer` вместо `Server`
- Инструменты через `registerTool()` вместо `setRequestHandler()`

---

### Проблема 2: server.tool не функция

**Ошибка:**
```
TypeError: server.tool is not a function
```

**Причина:** Метод `tool()` устарел (deprecated)

**Решение:** Использовать `registerTool()` с config объектом
