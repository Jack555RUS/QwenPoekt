## 🧠 ТЕКУЩЕЕ ПОВЕДЕНИЕ (задокументировано)

### Входные параметры:

```powershell
.\before-action-checklist.ps1
  -Action <string>        # delete, move, create, modify
  -Target <string>        # Цель действия (файл/папка)
  -Reason <string>        # Причина действия
  -Verbose                # Подробный вывод
  -Log                    # Логирование в файл
```

### Выходные данные:

```
- Консольный вывод (пошагово)
- Лог-файл (если -Log)
- Код выхода: 0 (успех) / 1 (отмена/ошибка)
```

### Зависимости:

```powershell
# Скрипт использует:
- git status --porcelain
- Get-ChildItem (поиск файлов)
- Select-String (поиск текста)
- Read-Host (ввод пользователя)
```

### Точки отказа:

```
1. Если _TEST_ENV/ не существует → предупреждение (не ошибка)
2. Если git не установлен → предупреждение (не ошибка)
3. Если цель не найдена → ошибка (для delete/move)
4. Если пользователь отменил → выход 0
```

### Улучшение 2: Оценка риска

**Новая функция:**
```powershell
function Get-RiskScore {
    param($Action, $Target)
    
    $score = 0
    
    # Факторы риска:
    $score += Get-DependencyCount $Target * 2      # Зависимости
    $score += Get-FileCriticality $Target * 3      # Критичность
    $score += Get-ChangeHistory $Target * 1.5      # История
    $score += Get-RollbackComplexity $Target * 0.5 # Откат
    
    return $score
}
```

**Вывод:**
```
🔴 ВЫСОКИЙ РИСК (15-25) — требуется подтверждение
🟡 СРЕДНИЙ РИСК (5-14) — рекомендуется подтверждение
🟢 НИЗКИЙ РИСК (1-4) — можно автономно
```

### Улучшение 4: Граф зависимостей

**Новая функция:**
```powershell
function Get-FileDependencyGraph {
    param($Path)
    
    # Поиск ссылок на файл
    $dependencies = Get-ChildItem $BasePath -Recurse -Include "*.md","*.ps1" |
        Select-String -Pattern [regex]::Escape($Path)
    
    return @{
        Files = $dependencies.Path | Select-Object -Unique
        Count = $dependencies.Count
        Risk = if ($dependencies.Count -gt 10) { "High" } else { "Low" }
    }
}
```
