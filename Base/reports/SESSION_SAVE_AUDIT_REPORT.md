## 🧠 КОРНЕВАЯ ПРИЧИНА

**Проблема:** Система автосохранения работает, но **не сохраняет диалог ИИ**.

**Почему:**

1. **MCP сервер не настроен правильно**
   - В `mcp.json` сервер `session-saver` указан, но не активирован
   - VS Code не подключает MCP сервер

2. **Автосохранение работает вхолостую**
   - Task Scheduler запускает скрипт каждые 2 минуты
   - Скрипт создаёт папку сессии
   - Но **диалог ИИ не передаётся** в скрипт

3. **Нет интеграции между ИИ и скриптом**
   - ИИ не знает о необходимости вызывать `/save-session`
   - Скрипт не имеет доступа к диалогу ИИ
