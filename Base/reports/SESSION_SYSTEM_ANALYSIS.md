## 🔄 ВЫЯВЛЕННЫЕ ДУБЛИКАТЫ

### 1. schedule-backup-tasks.ps1 vs create-tasks-manual.ps1

**Проблема:** Два скрипта для одной задачи

| Характеристика | schedule-backup-tasks.ps1 | create-tasks-manual.ps1 |
|----------------|---------------------------|-------------------------|
| **Назначение** | Создание задач Task Scheduler | Создание задач Task Scheduler |
| **Объём** | 400+ строк | 230 строк |
| **Задачи** | QwenPoekt-* (8 задач) | QwenPoekt-* (8 задач) |
| **Статус** | ✅ Полный | ⚠️ Дубликат |

**Рекомендация:** 
- ✅ Оставить `schedule-backup-tasks.ps1` (полная версия)
- ❌ Удалить `create-tasks-manual.ps1` (дубликат)

---

### 2. auto-save-chat.ps1 vs mcp-session-saver.js

**Проблема:** Оба сохраняют сессии, но по-разному

| Характеристика | auto-save-chat.ps1 | mcp-session-saver.js |
|----------------|-------------------|---------------------|
| **Метод** | Task Scheduler (2 мин) | MCP (прямой вызов) |
| **Данные** | Формирует сам | Получает от ИИ |
| **Качество** | Частичная переписка | Полная переписка |
| **Статус** | ⚠️ Резервный | ✅ Основной |

**Рекомендация:**
- ✅ Использовать `mcp-session-saver.js` как основной
- ✅ Оставить `auto-save-chat.ps1` как резервный (если MCP недоступен)

---

### 3. save-chat-log.ps1 vs mcp-session-saver.js

**Проблема:** Оба сохраняют переписку

| Характеристика | save-chat-log.ps1 | mcp-session-saver.js |
|----------------|------------------|---------------------|
| **Вызов** | PowerShell (вручную) | MCP (автоматически) |
| **Данные** | Передаются параметрами | Передаются через MCP |
| **Интеграция** | Отдельный скрипт | Интеграция с Qwen Code |
| **Статус** | ⚠️ Устаревает | ✅ Новый |

**Рекомендация:**
- ✅ Использовать `mcp-session-saver.js` для новой системы
- ⚠️ Оставить `save-chat-log.ps1` для обратной совместимости
