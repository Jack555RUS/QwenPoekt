## 📊 ЧТО ТАКОЕ "CONTEXT USED"?

**Значение:** Процент использованного **окна контекста** модели

---

## 🧠 АРХИТЕКТУРА КОНТЕКСТА

### 1. Окно контекста (Context Window)

**Определение:**
> Максимальное количество токенов, которое модель может "помнить" в одном диалоге

**Для Qwen Code (предположительно):**
- **Qwen-2.5-Coder-32B:** 128,000 токенов
- **Qwen-Max:** 32,000 токенов
- **Qwen-Plus:** 131,000 токенов

**Расчёт:**
```
16% от 128,000 = 20,480 токенов использовано
```

## ⚠️ ПОСЛЕДСТВИЯ ЗАПОЛНЕНИЯ КОНТЕКСТА

### 0-50%: ✅ Норма

**Симптомы:**
- ИИ помнит весь диалог
- Контекст полный
- Качество ответов высокое

**Действия:** Никакие

---

### 50-75%: ⚠️ Внимание

**Симптомы:**
- ИИ может забыть ранние сообщения
- Ответы становятся короче
- Может пропустить детали

**Действия:**
- Сохранить сессию (`/save-session`)
- Кратко резюмировать прогресс
- Рассмотреть новую сессию

---

### 75-90%: 🔴 Критично

**Симптомы:**
- ИИ забывает контекст задачи
- Повторяет ранее сказанное
- Ошибается в деталях
- Предлагает уже отклонённые варианты

**Действия:**
- **Срочно сохранить сессию**
- Начать новую сессию
- Восстановить контекст из `.resume_marker.json`

---

### 90-100%: ❌ Переполнение

**Симптомы:**
- ИИ "теряет нить"
- Ответы бессвязные
- Забывает названия файлов
- Не помнит предыдущие решения

**Последствия:**
- ❌ Потеря качества работы
- ❌ Ошибки в коде
- ❌ Дублирование работы
- ❌ Потеря токенов (пустые запросы)

**Действия:**
- **Немедленно начать новую сессию**
- Восстановить контекст
- Продолжить с новой

---

## 🔄 КАК ОСВОБОЖДАТЬ КОНТЕКСТ

### Способ 1: Новая сессия (рекомендуется)

**Команда:**
```
/stop или "начинаем новую сессию"
```

**Процесс:**
1. Сохранить текущую сессию
2. Закрыть VS Code (опционально)
3. Открыть заново
4. Прочитать `.resume_marker.json`
5. Продолжить

**Эффект:** 
- ✅ Контекст: 0% → ~5%
- ✅ Чистая память
- ✅ Высокое качество

---

### Способ 2: Резюме прогресса

**Команда:**
```
"Сделай резюме текущего прогресса для новой сессии"
```

**Что делает ИИ:**
1. Кратко описывает выполненное
2. Указывает текущий статус
3. Определяет следующий шаг
4. Сохраняет в `.resume_marker.json`

**Эффект:**
- ⚠️ Контекст: Не освобождается
- ✅ Но есть точка восстановления

---

### Способ 3: Закрыть файлы

**Действие:**
```
Закрыть лишние вкладки в VS Code
```

**Эффект:**
- ⚠️ Минимальное освобождение (~1-2%)
- ✅ Но снижает "шум" в контексте

---

### Способ 4: Очистить историю чата

**Действие:**
```
Удалить `.qwen/chat-history/` (вручную)
```

**Эффект:**
- ✅ Контекст: -10-20%
- ⚠️ Но ИИ забудет предыдущий диалог

**Риск:** Потеря контекста задачи
