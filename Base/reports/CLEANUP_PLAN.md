```powershell
# ❌ НЕПРАВИЛЬНО:
mkdir $output  # Если $output = "Структура папок создана"

# ✅ ПРАВИЛЬНО:
mkdir "TargetFolder"
```

## 📋 ПЛАН ИСПРАВЛЕНИЯ

### Этап 1: Очистка корня (30 мин)

**Скрипт:** `cleanup-root-mess.ps1`

```powershell
# Папки на удаление
$messFolders = @(
    "--name-only",
    "-r",
    "echo",
    "git",
    "ls-tree",
    "move",
    "⚠ Папка уже существует или ошибки",
    "✓ Папка reports создана",
    "54e32ed1d",
    "Папка создана или уже существует",
    "Перемещение завершено",
    "Структура папок создана",
    "Тестовый проект создан"
)

foreach ($folder in $messFolders) {
    $path = Join-Path $PSScriptRoot $folder
    if (Test-Path $path) {
        Remove-Item $path -Recurse -Force
        Write-Host "✅ Удалено: $folder"
    }
}

# Файл $null
$nullFile = Join-Path $PSScriptRoot '$null'
if (Test-Path $nullFile) {
    Remove-Item $nullFile -Force
    Write-Host "✅ Удалено: `$null"
}
```
