## 🔄 ВОССТАНОВЛЕНИЕ СЕССИИ

**PowerShell (автоматически):**
```powershell
.\resume-session.ps1
```

**Что делает:**
1. ✅ Читает .resume_marker.json
2. ✅ Копирует контекст в буфер
3. ✅ Показывает последнюю сессию
4. ✅ Вставьте в чат Qwen Code (Ctrl+V)

**Ручное восстановление:**
```powershell
Get-Content .resume_marker.json | Set-Clipboard
# Вставьте в чат Qwen Code
```

**Команда в чате:**
```
Восстанови контекст сессии (см. буфер обмена)
```

## ⚠️ ВАЖНО

**Qwen Code НЕ поддерживает автоматическое восстановление контекста!**

**Почему:**
- ❌ Qwen Code хранит историю в Qwen Cloud
- ❌ НЕ читает локальные файлы автоматически
- ❌ `/resume` показывает историю чатов, не контекст

**Решение:**
- ✅ Используйте `.\resume-session.ps1`
- ✅ Копирует контекст в буфер
- ✅ Вставляете в чат (Ctrl+V)
